フォルダ構成
> mkdir src
> mkdir src\components
> mkdir src\containers
> mkdir src\modules

モジュールセットアップ
 react-ts のreadmeとほぼ同じ

> npm init
> yarn add react react-dom @types/react @types/react-dom
> yarn add typescript awesome-typescript-loader source-map-loader --dev
> yarn add webpack webpack-cli webpack-dev-server --dev


> tsc --init

Reduxと関連モジュールのインストール
> yarn add redux @types/redux
> yarn add react-redux  @types/react-redux
> yarn add react-logger  @types/redux-logger
> yarn add react-thunk @types/redux-thunk
